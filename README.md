# OCR Berbasis Web
Salah satu kegiatan Aksi Bela Negara Pelatihan Dasar CPNS saya adalah membuat repo github OCR yang dapat digunakan semua orang. 
Proyek ini berguna untuk mengekstrak teks dari dokumen gambar atau PDF. Fitur ini mampu beroperasi secara offline dan tanpa perlu upload ke server, sehingga lebih cepat karena diproses di local.

## Tech Stack
- TailwindCSS
- Dropzone JS
- PDF.JS
- Tesseract JS

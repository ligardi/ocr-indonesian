OCR Online Bahasa Indonesia


OCR adalah Optical Character Recognitiion. Fitur OCR ini berguna untuk mengekstrak teks dari dokumen gambar atau PDF. Fitur ini mampu beroperasi secara offline dan tanpa perlu upload ke server, sehingga lebih cepat.


Dikembangkan sebagai salah sat aksi RABN CPNS BPS 2026.


Versi 0.2, April 2026

Masukkan File

Silakan masukkan file gambar atau PDF dalam kotak di bawah ini untuk diekstrak teksnya.

Klik untuk upload PDF atau drag ke dalam kotak ini.

(Hanya menerima file PDF untuk saat ini)

Hasil Ekstrak Teks

Teks ini yang kemudian akan disimpan di server dan digunakan oleh fitur archive search (Temu Kembali Informasi)

Daftar Halaman & Detail OCR

Detail halaman di bawah ini adalah fitur untuk keperluan demo dan Eksperimentasi.


Eksperimen OCR sebagai salah satu aksi Rencana Aksi Bela Negara (RABN)
